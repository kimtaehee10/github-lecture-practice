# 수업 실습 따라하기

## 실습 A — 첫 커밋

1. [github.com](https://github.com) 접속
2. 우측 상단 `+` → `New repository`
3. Repository 이름 입력 (영어, 예: `my-first-repo`)
4. **`Add a README file` ✅ 체크** ← 꼭 체크하세요!
5. `Create repository` 클릭
6. `Add file` → `Create new file`
7. 파일명: `hello.py`
8. 아래 코드 입력:

```python
# 나의 첫 번째 Python 파일
print("안녕하세요! 첫 커밋을 축하합니다 🎉")
print("GitHub는 코드를 저장하는 곳이에요.")
```

> ⚠️ GitHub는 코드를 실행하는 곳이 아니라 **저장하는 곳**이에요.

9. `Commit changes` 클릭 → 메시지 입력 → `Commit changes`
10. 시계 아이콘 클릭 → 커밋 히스토리 확인 ✅

---

## 실습 B — Fork & Pull Request

1. 선생님 실습 저장소 접속
2. 우측 상단 `Fork` 클릭 → 내 계정으로 복사됨
3. 내 Fork 저장소에서 `Contributors.md` 클릭 → ✏️ 아이콘
4. 아래 형식으로 내 이름 추가:

```
| 이름 | 한마디 | 
| 김북곽 | 오픈소스의 거인이 될거예요! | <--- 예시
```

5. `Commit changes`로 저장
6. `Contribute` → `Open Pull Request` → `Create Pull Request`
7. 선생님이 merge 해주면 내 이름이 공식 등록! 🎉
